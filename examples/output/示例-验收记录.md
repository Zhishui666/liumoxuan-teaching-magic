# 示例验收记录

## 验收对象

- 输入：`examples/input/示例-单份核心内容.md`
- 输出：`examples/output/示例-一备多用输出.md`
- Prompt 合约：`prompts/一备多用适配器-v0.md`

## 验收结论

本轮样例完成了“一份核心教学内容 -> 三个场景版本”的最小闭环，可以用于项目演示和后续真实材料替换。

## 可用点

- 三个版本都保留了同一个核心知识点：内容概括要围绕题目对象提取并合并关键信息。
- 三个版本的对象、场景和动作不同，不只是改写称呼或语气。
- 输出中保留了例题材料和可复核答案，便于教师检查是否偏离原始内容。
- 家长说明版避免堆叠教学术语，能用于课后反馈或家校沟通。

## 需改点

- 示例材料仍是项目内置样例，不等同于教师真实课堂材料。
- 目前只验证了语文学科中的一个现代文阅读知识点。
- 还没有记录教师实际修改耗时，暂不能量化减负比例。

## 下一轮优化点

- 用一份教师真实讲义或教案替换当前样例。
- 扩展到至少五个输出场景，加入复习清单版和课堂活动版。
- 在验收记录中增加“原始改写耗时”和“使用适配器后修改耗时”。
